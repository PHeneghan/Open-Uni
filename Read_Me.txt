The database should be named student.
When running the website start with the home page.
Registration leads on to form to collect registration information.
Student, brings you to a page with three options.
The Download page has a file that can be read or downloaded.
The Course Texts page displays a table of course texts taken from the database.
The Assignment page gives the option to upload a file.
The Tutor page has two options.
The Update Course Texts page allows more books to be added to the coursetexts table in the database.
The Update Results page, gives the options to insert new results, but is not functional beyond that.
The Courses page displays the modulesinfo view from the database.

The website does not make out bills, or a printable results page.
